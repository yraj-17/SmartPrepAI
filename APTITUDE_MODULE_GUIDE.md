# Aptitude Module Guide

## Student Flow

1. Login.
2. Open Dashboard.
3. Click Aptitude Mock Test.
4. Select difficulty.
5. Start test.
6. Attempt questions using A/B/C/D buttons.
7. Use Previous, Next, Mark for Review, Clear Response.
8. Submit manually or wait for timer auto-submit.
9. Check result analytics and AI feedback.

## Admin Flow

1. Login as admin.
2. Open Admin Dashboard.
3. Click Manage Aptitude Tests.
4. Add categories and difficulty levels.
5. Upload question images.
6. Create tests by selecting uploaded questions.

## Question Image Rule

Each question image should contain:

- question statement
- all options A, B, C, D

The database does not store the question text.
